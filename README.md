## Run Locally

To run this project locally -

Clone the project

```bash
  git clone https://github.com/rabbiincode/Founders
```

Go to the project directory

Install dependencies

```bash
  run - npm install to instals dependencies
```

Start the server

```bash
  run - npm run dev
```
